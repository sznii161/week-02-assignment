이원찬 (Lee Wonchan)

안녕하세요. 저는 컴퓨터소프트웨어공학을 전공하고 있는 이원찬입니다. 서버 운영, 네트워크, 백엔드 개발에 관심이 있으며 다양한 서비스를 직접 만들고 운영하는 것을 좋아합니다.

현재는 NodeJS, PHP, MariaDB, Redis 등을 활용한 웹 서비스 개발과 Linux 기반 서버 운영을 주로 하고 있습니다. 또한 Rocky Linux, Ubuntu Server 환경에서 미러 서버 구축, 로그 서버 운영, 네트워크 모니터링 시스템 등을 직접 구축해본 경험이 있습니다.

개인 프로젝트로는 암호화폐 모의투자 거래소인 Upcoin 서비스를 개발하고 운영하고 있으며, 실제 서비스 수준의 시스템을 만드는 것을 목표로 다양한 기능을 구현하고 있습니다.

앞으로는 네트워크 인프라와 대규모 웹 서비스를 안정적으로 운영할 수 있는 백엔드 및 서버 엔지니어로 성장하는 것을 목표로 하고 있습니다.
